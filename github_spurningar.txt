GitHub spurningar:

1) Hvað gera eftirfarandi git skipanir:

	git remote                    : Manage set of tracked repositories
	git remote add param1 param3  : Adds a remote named <name> for the repository at <url>
	git remote -v                 : Be a little more verbose and show remote url after name. 
	git push param1 param2        : Push commits made on your local branch to a remote repository.


2)  Hvað gera eftirfarandi git skipun:

	git pull param1 param2    : Performs a merge on the retrieved changes
	
3) Hver er munurinn á fork á GitHub og git clone?

"fork" creates a copy of the project hosted on your own GitHub account.

"clone" uses git software on your computer to download the source code and it's entire version history.
